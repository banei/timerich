## 一、核心理念与方法论

### 1.1 七条核心原则

1. **组合优于单品**：纳指100和红利低波负相关性显著，组合后波动率显著低于任一单品
2. **长期持有 ≥ 5年**，理想10年+，否则不要碰这套方案
3. **场外联接基金优先**，避开场内5-10%溢价陷阱
4. **定投而非一次性**，但定投强度可按估值信号调节
5. **每年再平衡一次**，这是免费的alpha
6. **多基金轮询+溢出阶梯**，应对QDII限购
7. **纪律 > 金额 > 技巧**——能坚持10年的60分方案，远胜执行不下去的95分方案

### 1.2 为什么组合而不是单品

| 单品视角 | 实际数据（2016-2026校对版） |
|---|---|
| 纳指100一次性10万元 → | 59.3万元，年化18.6%，最大回撤-34% |
| 红利低波一次性10万元 → | 22.7万元，年化8.2%，最大回撤-25% |
| **50/50组合（年度再平衡）** | **约41万元，年化14%，最大回撤约-22%** |

组合的真正价值不在收益最大化，而在**风险调整后收益最优**：少赚10万换取最大回撤从-34%降到-22%，绝大多数投资者扛得住的人数比例完全不同。

### 1.3 这套方案的alpha来源（按贡献排序）

| 来源 | 预计年化贡献 |
|---|---|
| 长期持有股票资产 | 7-9% |
| 估值信号调节投入强度 | +0.5-1% |
| 年度再平衡 | +1-2% |
| QDII限购溢出处理 | +0.5-1% |
| 场内场外费率优化 | +0.3-0.5% |
| **合计预期年化** | **9-13%** |

⚠️ **这套优化方案合计提升约2-4个百分点**——重要但不是革命性的。**真正决定你10年后赚多少的，是你能不能扛住中间的回撤不退出**。

## 二、心理预期管理

### 2.1 基于10年真实数据的合理预期

| 标的 | 过去10年年化（CNY） | 未来10年合理预期 | 触发条件 |
|---|---|---|---|
| 纳指100 | 18.6%（一次性）/ 19.1%（定投） | **8-12%** | 历史窗口期不会重演 |
| 红利低波 | 8.2%（一次性）/ 10.1%（定投） | **7-10%** | 红利风格周期性表现 |
| 50/50组合 | 约14% | **8-11%** | 与权重配置相关 |

### 2.2 必须接受的事实

**关于纳指100**：
- 过去10年是历史性的科技股窗口期（QQQ年化19%是1986年指数成立以来最强10年之一）
- 当前PE分位94%，未来3-5年大概率经历估值回归
- QDII限购短期内不会消失
- 汇率波动会影响人民币收益（过去10年人民币贬值9%放大了纳指收益约1pp/年，未来反转可能减损）
- **必然经历"扛不住想割肉"的时刻**——2022年纳指人民币计价回撤-26%，2026年初又回撤-15%

**关于红利低波**：
- 过去3年（2022-2024）连续跑赢沪深300是因为成长股熊市，不是常态
- 股息率6%+部分由股价下跌带来（典型如银行2024），不全是分红能力提升
- 行业集中度高：银行+煤炭+交运占比约60%，单一行业冲击会放大波动
- 在成长股大年（如2019、2020）会大幅跑输

**关于组合**：
- 50/50并不会"两边都赚"——会经历"两边都跌"的极端时刻（如2022年Q4）
- 再平衡不是"低买高卖"的精确操作，年内可能踏空
- 你会反复怀疑配比是否正确，**这种怀疑本身就是策略在起作用的副作用**

## 三、资产配置框架

### 3.1 三档配置模板（按风险承受度选一档）

| 类型 | 适用人群 | 纳指100 | 红利低波 | 债券/现金 |
|---|---|---|---|---|
| **进攻型** | 35岁以下，10年+不动用 | 50% | 30% | 20% |
| **平衡型** | 35-50岁，5-10年视角 | 35% | 40% | 25% |
| **防御型** | 50岁+，或5年内可能用钱 | 20% | 50% | 30% |

**选档判断**：年龄是参考，更重要的是问自己——"如果明天纳指部分跌40%，我会怎么想？" 如果是"加倍买入"选进攻型；如果是"算了忍住"选平衡型；如果是"我要赎回避险"选防御型。

### 3.2 总仓位上限

无论选哪档，**这套方案总额度上限是家庭金融资产的30%**。剩余至少：
- 6个月生活费 → 货币基金或活期理财
- 30%-40% → 其他资产（A股核心资产、港股、黄金、房产等）

**不要把它当唯一持仓**。指数会有连续5年跑不出超额收益的时候。

### 3.3 债券部分的简化方案

不用纠结选哪只债基。三选一：
- **国债ETF（511010）**：场内，0费率最低，约2-3%年化
- **纯债基金**（如易方达稳健收益110007）：场外，约3-4%年化
- **银行3-5年大额存单**：保本，约2-3%年化

债券部分的目标不是赚钱，是再平衡时的"弹药库"。

## 五、统一月度SOP

### 5.1 每月定投日：10号（避开月初月末额度紧张）

**Step 1：检查信号（5分钟）**

```
□ 查纳指100 PE分位（雪球/集思录搜"纳斯达克100"看估值）
□ 查中证红利低波股息率（中证指数官网搜H30269的factsheet）
□ 查USD/CNY汇率（参考但不主导决策）
□ 据此确定本月各类资产的"投入系数"（见Section 六）
```

**Step 2：计算本月各档应投金额**

| 资产 | 公式 |
|---|---|
| 纳指部分 | 月预算 × 目标权重 × 纳指系数 |
| 红利低波部分 | 月预算 × 目标权重 × 红利系数 |
| 债券/现金 | 月预算 × 目标权重 |

**例**：月预算5000元，平衡型（35/40/25），当月纳指系数0.5、红利系数1.0
- 纳指档：5000 × 35% × 0.5 = 875元 → **少投的钱不留现金，转投红利低波或债券**
- 红利档：5000 × 40% × 1.0 = 2000元
- 债券档：5000 × 25% = 1250元
- 调剂：少投的875元 → 加到红利低波或债券

**Step 3：执行纳指部分（关键的"溢出阶梯"）**

```
第1层：纳指联接前3只（161130/018043/270042）
   ↓ 按优先级依次买，限购就跳下一只
第2层：纳指联接后7只
   ↓ 还满不了
第3层：标普500联接（050025/118002）
   ↓ 还满不了
第4层：把多余资金加到红利低波或债券
```

**绝对不要**：把买不进纳指的钱留在货币基金"等额度放开"——历史经验是等不到。

**Step 4：执行红利低波部分**

- 月预算 ≥ 1万：50%场内563020 + 50%场外007466
- 月预算 < 1万：全部走场外007466（避免小额佣金占比过高）
- 场内ETF用**限价单**，挂在当日均价附近，不用市价单
- 场内分红收到后等3-5个交易日，按当时价买回（避开除权波动）

**Step 5：场内溢价检查（仅纳指ETF）**

| 溢价水平 | 操作 |
|---|---|
| ≤ 0.5% | 可以买场内（流动性好） |
| 0.5%-2% | 忽略场内，走场外联接 |
| > 2% | 场外正常买；若手上有场内ETF且溢价>5%，考虑卖出套利 |

**Step 6：记录到表格**

| 日期 | 标的 | 代码 | 金额 | 单价/净值 | 份额 | 系数 |
|---|---|---|---|---|---|---|
| 2026-06-10 | 纳指联接 | 161130 | 875 | 4.50 | 194.4 | 0.5 |
| 2026-06-10 | 红利低波 | 007466 | 2000 | 1.20 | 1666.7 | 1.0 |
| 2026-06-10 | 国债ETF | 511010 | 1250 | 110.50 | 11.3 | - |

## 六、估值信号与定投强度调节

### 6.1 纳指100：按PE-TTM分位调节

| 估值分位 | 月投系数 | 当前判断（2026/6）|
|---|---|---|
| < 30%（低估） | × 1.5 | |
| 30%-70%（合理） | × 1.0 | |
| 70%-90%（偏高） | × 0.7 | |
| **> 90%（高估）** | **× 0.5** | **当前在此区间** |

**查询位置**：理杏仁、雪球估值表、集思录都可以。看"近10年PE分位"。

### 6.2 红利低波：按股息率调节

| 股息率 | 月投系数 | 当前判断（2026/6）|
|---|---|---|
| ≥ 6% | × 1.5 | |
| 5%-6% | × 1.0 | 接近此区间（factsheet约4.7%）|
| 4%-5% | × 0.7 | |
| < 4% | × 0.3 | |

**查询位置**：中证指数公司factsheet（每月底更新）、雪球、集思录。

### 6.3 重要：调节不是择时

- 系数0.5 ≠ 暂停定投。**0.5也是定投**，只是金额少了一半
- 系数1.5 ≠ all in。是正常额度的1.5倍，剩余资金照常按月分批
- 系数永远在 0.5-1.5 区间，**绝不为0**

完全停投的人，往往再也不会买回来。降低强度比停掉强10倍。

### 6.4 系数变更的频率

- 每月查一次估值，每月决定下个月系数
- 同一档系数可能持续6-12个月不变（估值变化慢）
- **不要因为单日下跌就调高系数，也不要因为单日上涨就调低**

## 七、年度再平衡机制

### 7.1 时机

**每年12月最后一周**做一次。其他时候不操作（除非偏离极大）。

### 7.2 阈值

| 偏离幅度 | 操作 |
|---|---|
| 各资产偏离目标 ≤ ±5% | 不动，下年新投入按比例补回 |
| 偏离 ±5% ~ ±10% | 用次年前2个月的新投入资金来平衡（不卖出） |
| 偏离 > ±10% | 卖出涨多的，买入跌多的（真正再平衡）|

### 7.3 卖出顺序（如果需要卖出）

1. 先卖**持有超过1年**的份额（红利税0%，资本利得也更稳）
2. 优先卖**费率高的基金**（如联接A>ETF）
3. 同类基金中卖**总收益最低的**（节税）

### 7.4 实际案例演示

假设你2026年初是平衡型（35/40/25），账户里：
- 纳指：35万
- 红利低波：40万
- 债券：25万

2026年12月底（假设场景：纳指跌30%，红利涨10%，债券涨3%）：
- 纳指：24.5万 → 25%
- 红利低波：44万 → 45%
- 债券：25.75万 → 26%
- 总计：94.25万

实际配比 25/45/26 vs 目标 35/40/25 → 纳指偏离-10%，触发再平衡。

操作：
- 卖红利低波 4万（44万 × 9%）
- 卖债券 1万
- 买纳指 5万 → 纳指变成 30万（约32%，仍未到目标但接近）
- **不强求一次到位**，剩余偏差靠新投入补足

这样2027年纳指反弹时，你的纳指仓位会受益更多——**这是再平衡的免费alpha**。

## 八、应急情况处理

### 情况A：纳指QDII全部严格限购（每只只能买10-100元）

1. 把第1层基金的小额都买一遍（10只×限额）
2. 不够的部分**全部转标普500联接**
3. 标普还买不下 → 加到红利低波档（不要等纳指放开）
4. **绝对不要把钱留在货币基金**

### 情况B：基金公司突然放开大额（10万/日）

1. 这是难得的窗口，可以一次性把当月+下月预算都投完
2. **不要all in**——基金公司可能下个月又缩回去
3. 利用窗口期，但不要透支未来3个月以上的预算

### 情况C：场内ETF溢价 > 5%

1. 场外定投正常执行
2. 已持有的场内ETF可考虑卖出套利（同时申购场外）
3. 算清费用：卖场内佣金（0.0085%×2）+ 场外申购费（0.12%）+ 时间成本，溢价>3%才划算

### 情况D：纳指大幅回调（-20%以上）

1. 估值分位会自动从>90%降到70%以下，**系数自动从0.5升到1.0或1.5**
2. 这是定投的"红利期"，便宜筹码大量出现
3. **不要all in**，按系数1.5执行即可
4. 此时再平衡机制也会自动让你"卖红利买纳指"，叠加效果

### 情况E：红利低波连续跑输沪深300两年

1. **不切换策略**——红利风格本来就有周期性
2. 复盘是否成分股质量出问题（看年报）
3. 如果只是风格轮动，继续按计划执行

### 情况F：场内/场外基金清盘或合并

1. 提前1-2个月会有公告
2. 改投同类替代品（清单里有10只纳指、3只红利低波）
3. 不要因为产品变更而中断定投节奏

### 情况G：限购到完全暂停申购

1. 切换到列表中其他可申购的基金
2. 实在没有 → 当月暂停定投，资金转其他档（红利/债券）
3. 不要为了"必须投出去"而买不熟悉的产品

## 九、退出时机判断

### 9.1 合理的退出理由 ✅

1. **个人资金需求**：买房、结婚、教育、医疗、养老
2. **达到目标收益**：预设的5年/10年目标完成
3. **总仓位严重超配**：纳指占家庭总资产>40%（即使涨上去的）
4. **个人风险承受能力下降**：年龄增长、收入减少、家庭责任加重

### 9.2 不合理的退出理由 ❌

1. ❌ "已经涨了30%，落袋为安"——可能错过后面的100%
2. ❌ "新闻说要崩了"——新闻永远在喊崩盘
3. ❌ "想换个更好的产品"——大概率换错
4. ❌ "短期需要用钱"——说明一开始投入资金就不对
5. ❌ "看不懂为什么涨/跌"——长期持有不需要看懂短期波动
6. ❌ "别的资产/赛道在涨，我跑输了"——这就是FOMO

### 9.3 退出方式：分批

任何超过当年定投总额的卖出，都**分3-6个月卖完**，不要一次性。

### 14.3 重要警告

1. **过去10年是纳指的历史性窗口**（科技革命+QE+美元宽松+人民币贬值四重叠加），未来10年大概率回归8-12%
2. **过去5年是红利低波的相对强势期**（成长股熊市+利率下行），未来5年可能回到平庸甚至跑输大盘
3. **本手册的所有"系数""阈值""比例"都是经验值**，未来市场环境变化时需要review
4. **没有任何回测能预测未来**——这份手册的价值不在"最优化"，而在"可执行"

### 14.4 风险提示

- 单一指数集中度高，需配合其他资产做组合
- QDII额度政策、汇率政策、税收政策都可能变化
- 历史回测不代表未来表现
- 基金跟踪误差年化1-2%会真实减损收益
- 投资有风险，决策需谨慎，本手册不构成投资建议

### 14.5 每5年彻底review

这份手册基于截至2026年6月的数据和市场环境。任何投资方案都不是终身有效——**每5年（2031、2036、2041...）把这份手册拿出来，根据当时的市场、自身情况、产品变化做一次彻底review**。

具体review清单：
- 当前持有的所有基金是否仍存在且规模健康？
- 当前的费率结构是否仍是最优？
- 当前的配比是否仍符合自己的人生阶段？
- 是否有新的、更优的标的出现（如汇率对冲QDII）？
- 当前的估值信号阈值是否需要调整？
- 整个市场的核心叙事是否发生根本变化？
